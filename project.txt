SOUL PATH NUMEROLOGY - PROJECT DESCRIPTION
=============================================

OVERVIEW
--------
Soul Path Numerology is a faith-based web application that combines ancient numerology wisdom with biblical insights to help users discover their spiritual journey through sacred numbers. Built with Next.js 14 and modern React patterns, this application provides personalized numerology calculations with comprehensive interpretations that connect numerological meanings to biblical references, Tarot associations, and astrological alignments.

TECHNICAL STACK
---------------
- Framework: Next.js 14 (App Router)
- Language: TypeScript
- Styling: Tailwind CSS with custom color scheme
- UI Components: Radix UI primitives with custom styling
- Icons: Lucide React
- Font: Inter (Google Fonts)
- Build Tool: Next.js built-in bundler
- Package Manager: npm

-----------------

CORE FEATURES
-------------

1. NUMEROLOGY CALCULATOR
   - Interactive form for name and birthdate input
   - Real-time calculation of four core numerology numbers:
     * Soul Number (from vowels in name)
     * Outer Personality Number (from consonants)
     * Destiny Number (combination of Soul + Personality)
     * Life Lesson Number (from birthdate)
   - Detailed calculation explanations for transparency
   - Support for Master Numbers (11, 22, 33)

2. COMPREHENSIVE INTERPRETATIONS
   - Faith-based interpretations connecting numerology to biblical wisdom
   - Biblical references for each number meaning
   - Tarot card associations (Major Arcana 1-9)
   - Astrological planet alignments
   - Personal number vibration definitions (double delineation)
   - Master number special significance alerts

3. EDUCATIONAL CONTENT
   - Sacred Science page explaining numerology fundamentals
   - Biblical significance of numbers
   - Connections to Tarot and Astrology
   - Master number explanations
   - Calculation methodology details

4. RESPONSIVE DESIGN
   - Mobile-first approach with Tailwind CSS
   - Fixed header with mobile navigation drawer
   - Accordion-based results display for better organization
   - Custom color scheme: sage green (#4F5D4E), terracotta (#E07A5F), light gray (#F4F4F4)

CALCULATION ALGORITHMS
----------------------
- Letter-to-number conversion using traditional numerology mapping
- Vowel/consonant separation for Soul/Personality calculations
- Number reduction with Master Number preservation
- Birthdate calculation combining month, day, and year digits
- Comprehensive error handling and validation

PHASE 2: LINEAGE STORYBOARD COMPONENT
=====================================

OVERVIEW
--------
The next phase expands Soul Path Numerology into a comprehensive family lineage platform - essentially "Facebook for family history" with AI-powered storytelling capabilities. This component will allow users to create interactive family storyboards, record oral histories, upload photos, and use AI to transform raw materials into compelling family narratives.

CORE VISION
-----------
Transform family history preservation from static genealogy charts into dynamic, multimedia storyboards where:
- Family members can be added with relationships mapped
- Audio recordings capture oral histories and memories
- Photos and documents tell visual stories
- AI processes raw materials into polished narratives
- Interactive timelines show family evolution
- Stories connect to numerology insights for deeper meaning
